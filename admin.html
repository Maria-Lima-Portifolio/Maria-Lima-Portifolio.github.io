Acesso para publicar

Para publicar, você precisa colar aqui uma "chave de acesso" (token) do GitHub, gerada uma única vez.
1. Acesse github.com/settings/personal-access-tokens/new e faça login.
2. Em "Repository access", escolha "Only select repositories" e selecione o repositório deste site.
3. Em "Permissions", abra "Repository permissions" → "Contents" → selecione "Read and write".
4. Clique em "Generate token" e copie o código gerado (começa com "github_pat_...").
5. Cole o código abaixo e clique em Salvar. Ele fica guardado só neste navegador.
